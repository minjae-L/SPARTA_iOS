# CounterApp
### 기능
- "+" 버튼: 중앙 라벨의 숫자가 증가한다.
- "-" 버튼: 중앙 라벨의 숫자가 감소한다.
- "RESET" 버튼: 중앙 라벨의 숫자가 초기화된다.
- 중앙 라벨의 숫자가 10이 되면, "+" 버튼은 비활성화 된다.
- 중앙 라벨의 숫자가 -10이 되면, "-" 버튼은 비활성화 된다.


### UI
- 중앙 라벨은 CenterX, CenterY에 맞춤
- UIStackView를 사용하여 버튼 위치 배열
- NSLayoutContraint를 사용하여 버튼 위치 설정


## 개발 환경
- Xcode 15.2.0
